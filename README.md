# Практическая работа №8: Каталог товаров

## Цель работы

Создать веб-приложение "Каталог товаров" с использованием:
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: FastAPI (Python)
- **База данных**: SQLite3 с связанными таблицами

## Используемые технологии

| Технология | Назначение |
|------------|------------|
| **FastAPI** | Создание серверной части и API |
| **SQLAlchemy** | ORM для работы с БД |
| **SQLite3** | Хранение данных |
| **HTML/CSS** | Вёрстка и стилизация |
| **JavaScript** | Клиентская логика |
| **Uvicorn** | ASGI сервер |
